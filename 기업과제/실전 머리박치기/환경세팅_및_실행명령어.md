# 환경 세팅 및 실행 명령어

## 전체 순서

1. nigiri 설치 및 실행
2. pnpm install
3. arkd(가짜 오락실 서버) 실행
4. 코드 작성 및 테스트

---

## Step 0 — 리눅스 방(WSL) 입장하기

현재 열려있는 터미널에 **`wsl`** 이라고 세 글자만 치고 엔터를 누른다.  
터미널 맨 앞의 `PS C:\...` 글자가 초록색 이름이나 `$` 기호로 바뀌면 리눅스 방에 들어온 것이다.

---

## Step 1 — Nigiri 설치 및 실행

**Nigiri 설치** (WSL 리눅스 방 안에서 실행)
```bash
curl https://getnigiri.vulpem.com | bash
```

**가짜 오락실(arkd) 서버 켜기**
```bash
nigiri start --ark
```

**가동 중지**
```bash
nigiri stop
```

**데이터까지 완전 삭제 후 중지**
```bash
nigiri stop --delete
```

Nigiri는 가짜 비트코인 네트워크(regtest) 역할이다. 아직 Ark 서버 역할은 없다.

---

## Step 2 — 패키지 설치

새로운 터미널 창(nigiri 창 말고)을 열고 Arkade SDK 폴더 경로로 이동한 뒤:

```bash
pnpm install
```

---

## Step 3 — 테스트 실행

```bash
# 사전작업 (Alice → Bob 환경 구축 + fixture 저장)
npx tsx test/e2e/vtxo-verify/setup.ts

# 검증 테스트 실행
npx vitest run test/e2e/vtxo-verify/verify.test.ts
```

**잔고 확인**
```bash
docker exec -t ark ark balance
```

---

## 전체 개발 흐름

1. pnpm install → 의존성 설치
2. nigiri start --ark → 가짜 오락실 서버 실행
3. 검증 로직 코드 작성 (Tier 1: DAG 족보 재구성, 서명 검증, 온체인 앵커링 검증)
4. 테스트 실행
